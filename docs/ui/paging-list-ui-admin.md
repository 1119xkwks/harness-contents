# 페이징 리스트 UI (관리자)

관리자 페이지의 페이징 리스트 공통 레이아웃 및 동작 규칙을 정의한다.

---

## 전체 구조

```
┌─────────────────────────────────────────────────┐
│  검색 영역 (border로 구분선)                       │
│                                                   │
│  [검색조건1]  [검색조건2]  [검색조건3] ...          │
│                                                   │
│                                       [검색 버튼]  │
└─────────────────────────────────────────────────┘

Total 26                [옵션 버튼1] [옵션 버튼2] [추가]

┌─────────────────────────────────────────────────┐
│  페이징 리스트 (테이블)                            │
│  ─────────────────────────────────────────────── │
│  row 1                                           │
│  row 2                                           │
│  ...                                             │
│  row 10                                          │
└─────────────────────────────────────────────────┘

         [<] [1] [2] [3] [4] [5] [>]
```

---

## 1. 검색 영역

| 항목 | 규칙 |
|------|------|
| 위치 | 페이지 최상단 |
| 구분선 | border로 영역을 명확히 구분하여 가시성 확보 (필수) |
| 검색 조건 | text, select, checkbox, radio 등 페이지별로 다름 |
| 검색 조건 배치 | 검색 영역 상단에 가로 나열 |
| 검색 버튼 | 검색 영역 내 하단 우측에 배치 |
| 동작 | 검색 버튼 클릭 또는 Enter 키로 검색 실행 |
| 검색 시 | 페이지 번호를 1로 리셋 |

---

## 2. Total 건수 및 액션 버튼 영역

검색 영역과 테이블 사이에 위치하는 한 줄 레이아웃.

```
좌측: Total {건수}
우측: [특수버튼들(엑셀 등)] ... [추가(생성) 버튼]
```

| 항목 | 규칙 |
|------|------|
| Total 건수 | 좌측 정렬, 검색 결과 총 건수 표시 |
| 추가(생성) 버튼 | 우측 끝에 배치 |
| 특수 버튼 (엑셀 등) | 추가 버튼 왼쪽에 배치 |

---

## 3. 페이징 리스트 (테이블)

| 항목 | 규칙 |
|------|------|
| 기본 표시 건수 | 10건 |
| 건수 변경 | 셀렉트박스 제공 (10, 20, 50 등) |
| 헤더 정렬 | 없음 (컬럼 클릭 정렬 미지원) |
| 체크박스 | 없음 |
| 행 클릭 | 상세 페이지로 이동 |

---

## 4. 페이지네이션

| 항목 | 규칙 |
|------|------|
| 방식 | 번호 페이징 |
| 한 번에 표시할 번호 수 | 5개 |
| 이전/다음 버튼 | `<` / `>` 형태로 좌우 배치 |
| 위치 | 테이블 하단 중앙 정렬 |

---

## 5. 상태 처리

| 상태 | 표시 방식 |
|------|-----------|
| 로딩 중 | 스켈레톤 UI |
| 데이터 0건 | 테이블 영역에 "데이터가 없습니다." 메시지 표시 |
